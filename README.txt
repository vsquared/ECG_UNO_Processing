
Demonstrates using the ‘Processing’ app (v3.2.1) to display an electrocardiogram generated by an Arduino UNO and Olimex EKG/EMG Shield.

Processing app may be found at https://processing.org/download/

Reference:https://itp.nyu.edu/physcomp/labs/labs-serial-communication/serial-output-from-an-arduino/ (see included .pdf)

Settings for Olimex EKG/EMG Shield:

Ref E      Closed
3V/5V      5V
AIN SEL    A5(11,12 Closed)
D4/D9      D4
